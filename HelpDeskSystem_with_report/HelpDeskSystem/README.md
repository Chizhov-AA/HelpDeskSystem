# HelpDeskSystem

Учебный проект для технологической практики.

## Тема

Разработка информационной системы учета заявок службы технической поддержки предприятия на языке Python с использованием SQLite.

## Технологии

- Python 3
- PyQt5
- SQLite3

## Запуск

```bash
pip install -r requirements.txt
python init_db.py
python main.py
```

## Состав проекта

- `main.py` — графический интерфейс приложения.
- `database.py` — функции работы с базой данных.
- `init_db.py` — создание и первичное заполнение базы.
- `database/helpdesk.db` — SQLite-база данных.
- `sql/schema.sql` — структура базы данных.
- `sql/sample_data.sql` — тестовые данные.
- `docs/er_diagram.txt` — текстовое описание ER-диаграммы.

## Основные возможности

- просмотр пользователей;
- просмотр инженеров;
- просмотр заявок;
- создание новой заявки;
- назначение инженера на заявку;
- изменение статуса заявки;
- поиск и фильтрация заявок.
